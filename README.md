# KenOS docker image builder

### Builder for osdev compiling tools in docker.
##### It contains:
* i686-elf-gcc/++
* default gcc/g++
* binutils
* nasm
* grub-common
* curl (for bitbucket pipelines)

### Continous Integration:
There is also configured CI system for automatic deployment to Docker Hub. Enjoy!
